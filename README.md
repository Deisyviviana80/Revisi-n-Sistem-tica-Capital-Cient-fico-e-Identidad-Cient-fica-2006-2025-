# Revisi-n-Sistem-tica-Capital-Cient-fico-e-Identidad-Cient-fica-2006-2025-
Análisis bibliométrico de 115 artículos científicos recuperados mediante la API de OpenAlex (inglés y español, 2006–2025). Incluye limpieza de datos, análisis descriptivo temporal y geográfico, análisis de citaciones, acceso abierto y síntesis crítica de vacíos temáticos con perspectiva de equidad educativa.
